To run the server please use the following command: 

       java cs455.scaling.server.Server [portnum] [thread-pool-size]

 * portnum and thread-pool-size must be an integer      
       
       
To run the Client with the following command: 

      java cs455.scaling.client.Client [server-ip] [server-port] [message-rate]   

* server-port and message-rate must be an integer.  Message-rate is the number of messages to send per minute
